## 📋 项目概述

巳泷资源库是一款集影视、游戏、音乐、动漫、小说于一体的网络资源导航应用。采用现代化毛玻璃 UI 设计，旨在提供流畅的资源获取体验。

### ✨ 核心特性

* 🎯 **资源丰富**：全品类资源覆盖
* 🎨 **界面美观**：毛玻璃设计，现代化交互
* 🔍 **智能搜索**：多引擎实时定位
* 🎵 **媒体音乐**：全新重做，全网免费资源搜索
* 🎮 **游戏中心**：集成 Flash 与独立游戏，支持 Ruffle 播放器
* 🌐 **动态更新**：网页内容实时更新，无需打包发布
