# 梦幻聊天室 ✨

一个完全无服务器的实时聊天应用，支持私聊、群组和公共聊天，可上传任意大小的文件。

## 功能

- 📢 公共聊天室 - 所有人可见
- 👫 私聊功能 - 一对一私密聊天
- 👥 群组功能 - 创建群组聊天
- 📁 文件上传 - 支持任意大小的文件
- 🎨 梦幻紫罗兰主题 - 美丽的渐变效果
- 🔄 实时同步 - 使用 Socket.io 实时通信

## 安装和运行

### 本地运行

1. 安装依赖：
```bash
npm install
```

2. 启动服务器：
```bash
npm start
```

3. 打开浏览器访问：
```
http://localhost:3000
```

### 公网部署

你可以使用以下方式将应用部署到公网：

1. 使用 Vercel 或 Railway
2. 使用云服务器（AWS EC2, 阿里云等）
3. 使用任何 Node.js 托管服务

部署后，不同地区的人都可以通过你的服务器地址访问聊天。

## 使用说明

1. 打开应用后，会自动创建一个匿名用户
2. 在多个浏览器窗口/标签页中打开可以测试聊天
3. 点击「好友」标签添加朋友
4. 点击「群组」标签创建群组
5. 选择聊天，输入文字或拖拽文件，点击发送
6. 公共聊天里所有人都可以看到消息

## 技术栈

- 前端：HTML, CSS, JavaScript
- 后端：Node.js, Express
- 实时通信：Socket.io

## 注意事项

当前数据存储在内存中，服务器重启后数据会清空。生产环境建议使用数据库持久化存储。
